このアクショングループは、半導体材料の目標特性を設定し、最適化するための特性目標設定エージェントの機能を提供します。

以下の関数を含みます：

1. set_target_properties
   説明: ユーザーの要件に基づいて、半導体材料の目標特性を設定し、実現可能な特性範囲を提案します。
   パラメータ:
   - requirements: ユーザーから提供された材料要件の詳細説明。例: "高効率な太陽電池用半導体材料が必要"
     タイプ: string
     必須: はい
   - session_id: セッションを識別するための一意のID。指定しない場合は自動生成されます。
     タイプ: string
     必須: いいえ
   - user_id: リクエストを行ったユーザーのID。
     タイプ: string
     必須: いいえ

2. analyze_tradeoffs
   説明: 異なる材料特性間のトレードオフ関係を分析し、最適なバランスを提案します。
   パラメータ:
   - target_properties: 分析対象の目標特性セット。
     タイプ: object
     必須: はい
   - priority_feature: 優先すべき特性の名前。
     タイプ: string
     必須: いいえ
   - session_id: セッションを識別するための一意のID。
     タイプ: string
     必須: いいえ

3. validate_targets
   説明: 設定された目標特性が物理的・化学的に実現可能かどうかを検証します。
   パラメータ:
   - target_properties: 検証対象の目標特性セット。
     タイプ: object
     必須: はい
   - session_id: セッションを識別するための一意のID。
     タイプ: string
     必須: いいえ
