このアクショングループは、目標特性から最適な半導体材料の組成・構造を逆予測する材料逆設計エージェントの機能を提供します。

以下の関数を含みます：

1. design_materials
   説明: 目標特性に基づいて、最適な材料組成・構造を逆予測し、候補材料のリストを生成します。
   パラメータ:
   - target_properties: 目標とする材料特性のセット。
     タイプ: object
     必須: はい
   - session_id: セッションを識別するための一意のID。指定しない場合は自動生成されます。
     タイプ: string
     必須: いいえ
   - constraints: 材料設計の制約条件（コスト、毒性、希少元素の使用制限など）。
     タイプ: object
     必須: いいえ

2. rank_candidates
   説明: 生成された候補材料をさまざまな基準（目標特性との一致度、合成可能性、コストなど）に基づいてランク付けします。
   パラメータ:
   - candidate_materials: ランク付けする候補材料のリスト。
     タイプ: array
     必須: はい
   - ranking_criteria: ランク付けの基準と重み付け。
     タイプ: object
     必須: いいえ
   - session_id: セッションを識別するための一意のID。
     タイプ: string
     必須: いいえ

3. evaluate_feasibility
   説明: 提案された材料の合成可能性と安定性を評価します。
   パラメータ:
   - material: 評価対象の材料。
     タイプ: object
     必須: はい
   - session_id: セッションを識別するための一意のID。
     タイプ: string
     必須: いいえ
