このアクショングループは、半導体材料の検証実験を計画し、最適な実験条件を提案する実験計画エージェントの機能を提供します。

以下の関数を含みます：

1. create_experiment_plan
   説明: 提案された材料の検証実験計画を作成します。最小限の実験で最大の情報を得るための最適な実験設計を提案します。
   パラメータ:
   - materials: 検証対象の材料リスト。
     タイプ: array
     必須: はい
   - target_properties: 検証すべき目標特性のセット。
     タイプ: object
     必須: はい
   - session_id: セッションを識別するための一意のID。指定しない場合は自動生成されます。
     タイプ: string
     必須: いいえ
   - available_equipment: 利用可能な実験装置のリスト。
     タイプ: array
     必須: いいえ

2. optimize_conditions
   説明: 特定の実験の条件（温度、圧力、時間など）を最適化します。
   パラメータ:
   - experiment: 最適化対象の実験。
     タイプ: object
     必須: はい
   - optimization_goal: 最適化の目標（例：「収率最大化」、「純度最大化」）。
     タイプ: string
     必須: いいえ
   - session_id: セッションを識別するための一意のID。
     タイプ: string
     必須: いいえ

3. estimate_resources
   説明: 実験計画に必要なリソース（時間、材料、コストなど）を見積もります。
   パラメータ:
   - experiment_plan: リソース見積もり対象の実験計画。
     タイプ: object
     必須: はい
   - session_id: セッションを識別するための一意のID。
     タイプ: string
     必須: いいえ
